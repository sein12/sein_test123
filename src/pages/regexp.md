| 메서드                          | 설명                                     |
| ---------------------------- | -------------------------------------- |
| `pattern.test(str)`          | `str`가 패턴에 매치되면 `true`, 아니면 `false` 반환 |
| `str.match(pattern)`         | 매치된 부분의 배열 또는 `null` 반환                |
| `str.replace(pattern, repl)` | 패턴에 매치된 부분을 `repl`로 대체                 |
| `str.search(pattern)`        | 매치된 위치 인덱스(없으면 `-1`)                   |
